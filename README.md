## Run Locally

Clone this project 

```bash 
git clone https://github.com/AfriSplash-Remotely/AfriSplash-SuperAdmin.git
```

Cd into the project directory

```bash 
cd AfriSplash-SuperAdmin
```

Install dependencies
```bash
yarn  install
```

Run Project
```bash
yarn dev
```
